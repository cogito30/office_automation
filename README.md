# Office Automation

## Description
- 업무를 위한 기본 문서 도구 사용법

#### Document
- Word: Windows에서 사용하는 문서 편집 도구
- Pages: Mac에서 사용하는 문서 편집 도구

#### Presentation
- PowerPoint: Windows에서 사용하는 프레젠테이션 편집 도구
- Keynote: Mac에서 사용하는 프레젠테이션 편집 도구

#### SpreadSheet
- Excel: Windows에서 사용하는 스프레드시트 편집 도구
- Numbers: Mac에서 사용하는 스프레드시트 편집 도구
